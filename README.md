# Under the Hood Engineering

A small collection of technical notes and essays focused on understanding the systems beneath cloud-native tools.

## About

This repository is a personal space for exploring and documenting the underlying behavior of platform and infrastructure tooling, rather than just the tools themselves.

## Content

- `karpenter_vs_kubernetes_scheduler_blog_post.md` — an essay that explains the difference between the Kubernetes scheduler and Karpenter, and why the distinction matters.

## Why this repo exists

It is easy to learn what a tool does without learning how the underlying systems work. This repo is meant to keep that engineering foundation visible.

## How to use

Open the markdown file in a viewer or editor to read the essay. Use this repository as a reference when studying Kubernetes scheduling, autoscaling, and cloud infrastructure behavior.

## Contributing

If you want to add more essays or notes, add a new markdown file with a clear title and summary.
